# Job Application Tracker CLI

A command-line based job application tracking system built with Python and MySQL.

This project allows users to manage job applications, export data, and perform exploratory data analysis (EDA) with visualization.

---

## Features

- Add, update, delete job applications
- Search by company or position
- Application statistics (status distribution, response rate)
- CSV export functionality
- Exploratory Data Analysis (EDA)
- Salary distribution visualization
- Acceptance rate analysis across salary ranges

---

## Tech Stack

- Python
- MySQL
- Pandas
- Matplotlib
- Argparse
